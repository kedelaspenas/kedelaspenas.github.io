Research

Extending Protein Tracking to 3D using Asigmatism, Gaussian Fitting, and CNN
Rotation Project
In this project, we presented and contrasted two approaches (gaussian fitting with CNN-based denoising and CNN axial trajectory trend classification) extending the analysis of protein movement to three dimensions by employing astigmatism.

Cell Detection and Instance Segmentation in a Memory-Constrained Environment using YOLO, UNet, DeepLab, and Conditional Random Fields
Rotation Project
This project examined different deep learning techniques for cell detection and segmentation on resource-constrained environments such as the NVIDIA Jetson TX2.

Plasmodium Parasite Detection and Species Identification on Thin Blood Smears using Shape Feature Analysis and Convolutional Neural Networks
MSc Research Project
This research proposes an intelligent system that automates the diagnostic process for malaria by detecting Plasmodium parasites and the specific species on images of thin blood smears using convolutional neural networks.

eHEART: Electronic Heart Failure Evaluation using Bayesian Networks
Undergraduate Project
This study proposes a system which gives the probabilities of developing heart failure in years to come given the patient's current condition using a Bayesian network of risk factors and their corresponding probabilities.
